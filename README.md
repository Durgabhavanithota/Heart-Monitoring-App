# Heart Monitoring App

A health monitoring and emergency alert application designed to analyze heart rate conditions and assist users during emergencies.

## Project Status

🚧 Currently under development

## Features Implemented

* Heart rate analysis
* Emergency alert logic
* Emergency contacts management
* Dark mode
* Multilingual support
* Mobile-friendly UI

## Features In Progress

* SMS integration
* Email alert integration
* Live location sharing

## Technologies Used

* Readdy AI
* Responsive UI Design
* API/Webhook Integration Concepts
